## Descripción del programa
El programa será elaborado en C++, su archivo principal main.cpp asegurará que se muestre un menú con opciones y se ejecuten las operaciones matemáticas según la selección del usuario en consola.
Las opciones a elegir son:
1. Suma
2. Resta
3. Multiplicación
4. División
El resultado de la operación se mostrará con el formato "Resultado: " y se evaluarán opciones invalidas.
